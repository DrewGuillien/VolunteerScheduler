Admin

As an admin you can do everything a Program Manager and Volunteer can in addition to: managing users, generating reports, and finalizing schedules!

Creating a New User

  1. Navigate to the Admin Dashboard in the upper-left hand corner
  2. Select the "Create User" button
  3. Fill out all fields for the user and select their role
  4. Once ready, click the blue "Create" button and the user will be instantly adde to the database! Wow! Technology!

Deleting/Deactivating a User

  1. Navigate to the Admin Dashboard in the upper-left hand corner
  2. Select the "Manage Users" button
  3. Here a table will populate with all users
  4. Each row represents a user. Now you can select to enable/deactivate a user as well as the option to delete them.

Finalizing a Schedule

  1. Navigate to the Programs view by clicking "Programs" on the navigation bar
  2. Find the program you wish to finalize and click the blue "Finalize" button
  3. Once the program is finalized, an email will be sent out to all users notifying them. The Edit and Finalize buttons will now also be disabled.

Program Manager

As a Program Manager you can do everything a Volunteer can in addition to: managing programs and activites!

Creating a Program

  1. Navigate to the Programs view by clicking "Programs" on the navigation bar
  2. Click the "Create New Program" button to pull up a pop-up window
  3. From here you must enter a title and description.
  4. You can also add activities by clicking the "Add Activity" button which will popualte additional fields for you to modify. Each activity has at least one shift that can be modified.
  5. Once you are content with your selections, click the blue "Add New Program" button at the bottom of the window to complete the program creation.

Editing a Program

  1. Navigate to the Programs view by clicking "Programs" on the navigation bar
  2. Find which program you wish to edit and click on the white "Edit" button.
  3. Once you are content with your changes, click the blue "Save Changes" button.

Removing a Program

  1. Navigate to the Programs view by clicking "Programs" on the navigation bar
  2. Find which program you wish to remove and click on the red "Remove" button.
  3. The program, activites, and shifts will now be removed from the system.

Volunteer

As a Volunteer you can view all available programs, activities, and shifts available!

Shift Sign-Up

  1. Navigate to the Programs view by clicking "Programs" on the navigation bar
  2. Find which program you wish to sign-up for and click on the green "View" button.
  3. The activity list will now appear. Select which activity you want to sign-up for by selecting the green "Volunteer" button.
  4. A pop-up window will now be displayed with the available shifts. Select one and click on the blue "OK" button to register.
